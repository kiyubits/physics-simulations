# Double Pendulum

ok, so the double pendulum is essentially one pendulum that is connected to the bob of another pendulum, which is connected to a fixed point. there is no closed solution for this, like the normal pendulum, because we get an ODE that is non-solvable, the closest we can get is using a small angle approximation.

I'm not sure if there's a way to get a solution using Newtonian mechanics, but I've worked with finding the form using Lagrangian/Hamiltonian methods, I'll put it in the blog probs.

However, I know that it also uses small angle approximations, so there is value in using computation to simulate the motion of it because there is no general closed solution. I'm curious as to how it ties into chaos theory.
